# 💾 PC 설치파일 관리자

PC에 설치된 프로그램을 한눈에 확인하고, 중복 파일을 탐지해 저장공간을 효율적으로 관리할 수 있는 데스크탑 앱입니다.

---

## ✨ 주요 기능

### 📦 설치 프로그램 관리
- Windows 레지스트리 + Program Files + Microsoft Store 앱까지 전체 스캔
- 프로그램 아이콘, 버전, 제조사, 설치 날짜, 크기, 마지막 실행일 표시
- 현재 실행 중인 프로그램 실시간 표시 🟢
- ⚠️ 불필요한 번들앱/보안앱 자동 감지
- 체크박스로 다중 선택 후 삭제
- 프로그램 우클릭 → 바로 실행 / 삭제
- 더블클릭 → 설치 경로 탐색기로 열기
- 이름/크기/날짜 기준 정렬 및 검색

### 🔁 중복 파일 탐지
- 문서, 다운로드, 바탕화면, 사진 폴더 자동 스캔 (폴더 추가 가능)
- MD5 해시 + 바이트 비교로 정확한 중복 탐지
- 경고 레벨 자동 분류
  - ✅ 일반 중복 — 안전하게 삭제 가능
  - ⚠️ 다른 폴더 / 실행파일 — 각각 필요할 수 있음
  - 🚨 파일명 다름 / 내용 불일치 — 삭제 주의
- 🔁 중복 선택 버튼으로 원본 1개 남기고 자동 선택
- ⚙️ 설정: 안전 모드 / 전체 모드 / 경고 알림 on·off

---

## 🖥️ 실행 환경

- **OS**: Windows 10 / 11

---

## ⚙️ 실행 방법

### 🟢 EXE 파일 (설치 불필요)

1. `PC설치파일관리자.exe` 다운로드
2. 다운로드한 `PC설치파일관리자.exe` 더블클릭
3. 관리자 권한 요청 창(UAC)이 뜨면 **예** 클릭

> Python이나 별도 라이브러리 설치가 필요 없어요!

## 📖 사용법

### 📦 설치 프로그램 탭
1. 앱 실행 시 자동으로 스캔 시작 (로딩바로 진행률 확인 가능)
2. 검색창에 프로그램 이름 입력해 빠르게 검색
3. 정렬 드롭다운으로 이름 / 크기 / 날짜 / 불필요 앱 기준 정렬
4. 삭제할 프로그램 체크박스 선택 후 **🗑️ 선택 삭제** 클릭
5. 프로그램 우클릭 → **▶️ 프로그램 실행** 또는 **🗑️ 프로그램 삭제**
6. 더블클릭 → 설치 경로를 탐색기로 열기
7. **🔄 다시 스캔** 버튼으로 목록 새로고침

### 🔁 중복 파일 탭
1. **🔍 중복 파일 스캔 시작** 클릭 (로딩바로 진행률 확인 가능)
2. 스캔 완료 후 중복 파일 목록 확인
3. 범례 참고해 경고 레벨 확인
   - ✅ 표시 없음 — 안전하게 삭제 가능
   - ⚠️ 주황색 — 다른 폴더 또는 실행파일, 확인 후 삭제
   - 🚨 빨간색 — 파일명 다름 또는 내용 불일치, 삭제 주의
4. **🔁 중복 선택** 클릭 → 원본 1개 남기고 자동 선택
   - 다시 클릭하면 선택 취소
5. 직접 체크박스로 개별 선택도 가능
6. **🗑️ 선택 파일 삭제 (N)** 클릭해 삭제
7. 더블클릭 → 탐색기에서 파일 위치 열기
8. 우클릭 → **📂 파일 위치 열기** 또는 **🗑️ 파일 삭제**
9. **📁 폴더 추가** 로 스캔 범위 확장 가능

### ⚙️ 설정 (중복 파일 탭)
- **🛡️ 안전 모드** (기본): ⚠️ 🚨 파일은 중복 선택에서 자동 제외
- **⚡ 전체 모드**: 경고 파일 포함해 모두 선택
- **경고 알림**: 위험 파일 삭제 시 경고창 표시 여부

---

## ⚠️ 주의사항

- 삭제한 파일은 **휴지통으로 이동하지 않고 즉시 삭제**됩니다. 신중하게 선택하세요.
- 언인스톨러가 없는 프로그램(일부 게임 런처 등)은 자동 삭제가 불가능합니다. 해당 런처를 통해 직접 삭제해주세요.
- Microsoft Store 앱 삭제는 PowerShell 명령어를 통해 실행됩니다.
